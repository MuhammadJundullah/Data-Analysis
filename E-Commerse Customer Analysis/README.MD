# Repositori Analisis Data Pelanggan Toko

Repositori ini berisi analisis data pelanggan dari toko fiktif. Tujuan dari analisis ini adalah untuk memahami karakteristik pelanggan ideal dan memberikan wawasan yang dapat ditindaklanjuti untuk meningkatkan strategi bisnis.

## Dataset

Dataset yang digunakan dalam analisis ini adalah data sintetis yang tersedia di Kaggle. Dataset ini mencakup informasi tentang demografi pelanggan, pendapatan tahunan, pengalaman kerja, ukuran keluarga, dan skor pengeluaran.

## Persyaratan

Untuk menjalankan kode dalam repositori ini, Anda perlu menginstal pustaka berikut:

- `pandas`
- `numpy`
- `matplotlib`
- `seaborn`

Anda dapat menginstal pustaka ini menggunakan pip:

```bash
pip install pandas numpy matplotlib seaborn
```

## Cara Menggunakan

1. **Klon repositori** ini ke komputer lokal Anda.
2. **Buka notebook Jupyter** `Analisis_Data_Pelanggan.ipynb`.
3. **Jalankan sel kode** secara berurutan.

## Wawasan

Analisis ini mengungkap beberapa wawasan penting tentang pelanggan toko:

- Rata-rata pelanggan memiliki pendapatan tahunan lebih dari $50.000.
- Pelanggan cenderung memiliki pengalaman kerja 0-2 tahun.
- Ukuran keluarga rata-rata pelanggan adalah 2.
- Skor pengeluaran dipengaruhi oleh pendapatan tahunan dan ukuran keluarga, tetapi tidak secara signifikan.
- Pelanggan dengan ukuran keluarga 4 memiliki skor pengeluaran tertinggi, tetapi tidak jauh berbeda dengan ukuran keluarga lainnya.
- Pelanggan dengan pengalaman kerja 2-3 tahun memiliki skor pengeluaran rata-rata yang relatif tinggi.
- Usia pelanggan tidak memengaruhi skor pengeluaran.
- Pelanggan perempuan 20% lebih banyak daripada pelanggan laki-laki.
- Pelanggan laki-laki cenderung memiliki skor pengeluaran rata-rata yang lebih tinggi.
- 32,1% pelanggan paling banyak berprofesi sebagai Artis, diikuti oleh Kesehatan sebesar 17,1%.
- Pelanggan yang berprofesi di bidang Hiburan memiliki skor pengeluaran rata-rata tertinggi.

## Kesimpulan

Analisis ini memberikan pemahaman berharga tentang pelanggan ideal toko. Wawasan ini dapat digunakan untuk mengembangkan strategi pemasaran yang lebih terarah, meningkatkan pengalaman pelanggan, dan mendorong penjualan.
